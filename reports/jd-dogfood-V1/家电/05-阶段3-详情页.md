## 京东家电场景关键观察(对比 3C 阶段 3)

### 1. **空调 vs 3C 规格表结构差异(关键发现)**
- **3C (iPhone 16) `.attrs` 内部结构**:
  ```html
  <div class="attrs">
    <div class="item ">                  ← class 末尾有空格
      <div class="label"><span class="text"> 品牌</span></div>   ← LABEL
      <div class="value"><div class="text">Apple</div></div>     ← VALUE
    </div>
    ... (29 项)
  </div>
  ```
- **空调 (海尔 KFR-35GW/E1-1) `.highlight-attrs` 内部结构**:
  ```html
  <div class="highlight-attrs">
    <div class="item">                    ← class 末尾无空格
      <div class="title text-ellipsis" title="大1.5匹">大1.5匹</div>   ← VALUE
      <div class="desc">
        <div class="text text-ellipsis" title="匹数">匹数</div>          ← LABEL
      </div>
    </div>
    ... (3 项)
  </div>
  ```
- **核心差异**: 3C 是 `.label .text` + `.value .text`(label-value 顺序),空调是 `.title` + `.desc .text`(value-label 顺序) - **值/标签颠倒!**
- **vortex selector 策略**: 跨品类详情页不能写死 label/value 位置,需先探查实际结构。D5 编程调用应做"先探查结构再选 selector"两步走,不能用单一通用 selector

### 2. **空调详情页完整规格表是 SPA AJAX 异步渲染(对比 3C 立即渲染)**
- **3C 阶段 3**: `.attrs` 29 项在 DOM 加载完毕即可见(SSR 服务端渲染,首次 observe 即可提取)
- **空调阶段 3**: `.attrs[0]` 是空容器 `<div class="attrs"></div>`,需要 click "商品详情" tab 或"查看全部参数" 触发 AJAX,真实规格数据才填充
- **本环境实测**: click "商品详情" tab 后 `.attrs[0]` 仍为空(网络无数据返回,京东 SPA 渲染未完成) - **D5 selector 验证通过(空容器存在),数据缺失是 SPA 渲染问题**
- **vortex 跨品类评测策略**: 完整规格表应**等 AJAX 完成后提取**,不能像 3C 一样初次 observe 即读取。`vortex_wait_for custom` 条件可设 `document.querySelectorAll('.attrs .item').length > 5`

### 3. **空调 vs 3C SKU 树结构差异(关键发现)**
- **3C (iPhone 16) SKU 树**(4 级): 系列品 (iPhone 14/15/16/16 Plus) + 颜色 (5 色) + 存储容量 (128GB) + 购买方式 (公开版/京补版)
- **空调 (海尔) SKU 树**(2 级): 系列品 (5 项:省电爆品/省电升级/舒适风/健康风/套装) + 款式 (10 项:大1.5匹/大1匹/大3匹/2匹 等)
- **核心差异**: 3C SKU 关注"硬件规格"(版本+颜色+容量),空调 SKU 关注"产品系列+匹数"(系列+适用面积)
- **vortex 评测策略**: 跨品类 SKU 树必须先 observe 实际选项,不能预设层级数。3C 的 4 级和空调的 2 级都需要"先 observe 再 click"

### 4. **空调 vs 3C 优选服务差异(关键发现)**
- **3C 优选服务** = AppleCare 维修类(AppleCare 1年碎屏电池¥299 / AppleCare+ 1年¥649 / 2年AppleCare+¥1199 / AppleCare 2年碎屏电池¥589)

- **家电特色**: 空调"系列品+款式"二级 SKU + 延保/换新服务 + 配送上门安装 + 价保 618,3C 完全没有这类服务
