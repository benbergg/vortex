## 京东 3C 评价阶段独有发现

### 1. 评价结构 BEM 风格,与规格表共用 `.item` 类

**问题陷阱:** 详情页 `.item` 类被规格表和评价列表共用。
- 规格表: `.attrs > .item` (Phase 4.3 D5)
- 评价列表: `.comment-root ul.list > li.item` (本次)

**正确 selector:**
```javascript
// 评价容器(本次评测 root)
'.comment-root'

// 评价列表(UL,3 条评价)
'.comment-root ul.list'

// 单条评价(LI)
'.comment-root ul.list > li.item'

// 评价内容(2 行折叠)
'.comment-root ul.list > li.item .info'

// 评价用户(3 字内显全文,3+ 字中间 ***)
'.comment-root ul.list > li.item [class*="user"], [class*="name"]'

// 标签云
'.comment-root .tags > div'

// 全部评价入口
'.comment-root .all-btn'

// 顶部 95% 好评率
'.everyone-reviews'
```

### 2. 京东用户隐私打码规则

- 用户名 ≤ 3 字: 显全文(吴***焱 / 我***c)
- 用户名 > 3 字: 中间 `***` 替代(测试样本不够,需复跑确认)
- 评价文本: **完整可见**,无打码

### 3. 评价扩展机制(SPA 锚点)

点击"全部评价"不会跳转 URL,京东采用 **SPA 锚点展开**:
- 点击前: `.comment-root` 含 2 个 block (top-block + 标签云)
- 点击后: `.comment-root` 含 2 个 block (top-block + mt24[标签云+列表+全部评价入口])
- **评价区没有 URL 变化**,无法用 URL 判断状态

### 4. 评价项缺日期字段

`[class*="date"]` / `[class*="time"]` 全部 0 命中。京东 3C 评价页**不展示评价日期**(可能折叠到点开详情才看),需后续阶段确认。

### 5. 视频评价缺失

本次 3 条评价**全部为纯文本**,无 `<video>` / `[class*="video"]` 元素。hasVideo=false。
**有图片缩略图**(`[class*="photo"], img[src*="jfs"]` 命中),本次样本评价无图但评价区整体有图片评价存在(photoImgs 全文档 92 命中)。

### 6. NOT_STABLE 频繁触发

评价区位于 scrollY=807-1303 区间,该区间有左侧 sticky 导航 + 顶部 CSS transition,导致 click 动作频繁失败。
**所有 vortex_act 调用必须 options={force:true} 兜底**,否则 5s timeout 后 NOT_STABLE。
