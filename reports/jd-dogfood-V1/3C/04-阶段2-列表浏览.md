## 京东场景关键观察

1. **空名率与 Phase 1 完全吻合** (29.65% vs 29.90%) — 京东 DOM 结构性特征而非偶发。**根本原因**(由 Phase 1 已确认):京东 logo 和客服图标(img/svg 渲染,accessibility tree 无 name)。本次列表页 59/199 空名中,e10/e14/.../e70 的 link 大多对应客服图标,button 大多对应加购车按钮(图标无 name)。**意义**:京东场景的 a11y 缺陷是稳定的、可预期的;vortex 在京东场景下,大量"未命名 link/button"是结构性事实,**不应视为 vortex 缺陷**。

2. **D3 降级原因新类型** — Phase 4.1 D3 ⚠️ 的原因是 NOT_STABLE(顶部搜索栏 CSS transition),**本次 D3 ⚠️ 的原因是"无 ref 可点"**(firstCard 是 div 不在 observe 列表,卡内唯一 a 是客服链接,唯一 button 是加购车)。两种降级原因不同但都可降级通过。**京东新搜索页的 click 模式 = 整张 div onclick**,与淘宝/天猫的"明确 a 链接跳转"模式根本不同。**vortex 后续优化建议**:vortex_observe 应考虑将带 onclick + cursor:pointer 的 div 暴露为可点击 ref(避免后续阶段类似降级)。

3. **firstCard.onclick 桩函数 `function kd(){}`** — React 合成事件的标志(React 把 onclick 重新绑定到 root,element.onclick 是桩)。这意味着:
   - `element.click()`(编程派发)不能触发 React 跳转,必须真实 mouse 事件
   - 必须用 `vortex_mouse_drag` 或 `vortex_evaluate` 调用 React 内部事件 API(本次未尝试,留作后续)
   - **vortex_evaluate 触发 click 已验证无效** — 这是 React 框架通病,非京东特有

4. **京东新搜索页 React 重写特征汇总**:
   - 商品卡 class: `_card_<hash>_<line>`(CSS Module 风格,hash 每次构建变,不能作稳定 selector)
   - 父级有 `data-pointId` / `data-sku`(用于埋点)
   - 卡内结构:1 `<a>`(客服) + 1 `<button>`(加购车) + N `<div>`(价格/标题/促销文案) + 多 `<img>`(商品图/标签图/装饰图)
   - **无显式商品详情 `<a>`**,跳转全靠 firstCard.onclick
   - 图片 `alt` 大多为空(自营标签是唯一带 alt 的),无障碍支持差

5. **京东自营 class 标志全部失效** — `[class*="self"]` / `[class*="ziying"]` / `[class*="flag"]` 都没命中。**唯一可靠识别方式是 `document.body.textContent.includes('京东自营')` 文字判断**(或限定到商品卡内 textContent 包含"京东自营")。这与 Phase 0 描述的"京东自营 logo 是 img,无 class"一致;**Phase 4.1 D2 没在搜索结果页上下文出现此问题是因为搜索结果页没显示商品卡,只显示搜索框**。

6. **D7 waitedMs=1 + D8 waitedMs=0** — 京东新搜索页 SPA 加载和跳转都极快(vite/React.lazy + SSR),与 Phase 4.1 一致。**D8 waitedMs=0 比 Phase 4.1 还要快**(Phase 4.1 D7 waitedMs=0 是搜索跳转,本次 D8 waitedMs=0 是商品卡跳转) — 京东 SPA 跳转性能极佳。

7. **`_addCart_65r2s_14` 新模式** — 列表页加购车按钮 class 是 `_addCart_<hash>_<line>`(CSS Module 风格),与 Phase 0 描述的"加入购物车 `<div id="add-to-cart">`"并存(详情页是 id 选择器,列表页是 class 选择器)。**Phase 0 的"加入购物车" id 模式在列表页不存在,列表页加购车是 class + button**。后续 04 阶段(详情页加入购物车)用 `#add-to-cart`,后续若需列表页加购车则用 `[class*="_addCart_"]` + button(留作记录)。

8. **D2 extract maxLength=5000 截断提示** — 列表页文本量 6646 字节,5000 上限触发 VORTEX_TRUNCATED 警告。**建议** vortex_extract 在京东列表页应支持 maxLength=8000(给后续阶段兜底),或调用时分批 extract(每段 maxLength=4000 滚动一次)。本次截断不影响关键字段召回(价格/评价/自营都在前 5000 字节)。
