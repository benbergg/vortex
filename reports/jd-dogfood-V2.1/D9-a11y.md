## V2.1 D9 结论

### P0 修 3 端到端验证 = ✅ 完美
1. vortex_observe 在京东 3C 详情页 main frame 0 不再 "frame 0 not scanned", **返 73 真实元素**
2. 滚动 2000 后仍返 **54 元素**, 覆盖 sticky 交互 (SKU/颜色/加购/购买)
3. 元素类型完整: link / button / textbox / div / span (含 reactClickable 标记)
4. 真实元素 refs: e0-e72 (京东 3C 详情页完整 DOM 树)

### D9 a11y 真实数据 (V2.1 唯一)
- aria-label coverage: **13.43%** (京东 SPA 不主动加 aria-label, 依赖 textContent fallback)
- accessible-name coverage: **95.52%** (textContent 覆盖 95.52% 元素, aria 不必需)
- role 实际使用: navigation (顶导) / button (加减) / menu (顶栏) / menuitem (顶栏 item)
- 与 V2 重测版 aria 数据一致 (V2.1 修复 observe, 不修 SPA 真实 a11y)

### D9 评测意义
- V2 重测版 "frame 0 not scanned" 是 **observe 工具 bug**, 不是京东 SPA 缺 a11y
- V2.1 P0 修 3 修后, observe 真实数据 = 京东 SPA 实际 a11y 水平 (13.43% aria, 95.52% acc)
- **V2.1 D9 = D9 评测数据生效的真正起点**
