## V2.1 D11 结论

### D11 动效真实数据
- **13 sticky 元素** (顶导 / left-tabs / page-right / activity-banner / login-bottom-bar)
- **114 animated 元素** (logo-icon / area-mini / iconfont + 111 其他)

### P0 修 3 关联
- V2 重测版 observe 返空, **D11 sticky/animated 不可测** (因为 observe 拿不到 DOM 元素)
- V2.1 P0 修 3 修后, observe 返真实 DOM, **D11 真实数据**:
  - sticky: 13 个 (京东详情页右栏 + left-tabs-nav + login-bottom-bar + activity-banner)
  - animated: 114 个 (京东 icon 大量 + SPA 路由动画)

### D11 评测意义
- 京东 SPA 大量 sticky (顶/侧/底) + 动画, LLM 驱动自动化需:
  - vortex_act click on 京东 sticky bar 元素 (加购/购买) 需 force=true (V2.1 已验证)
  - vortex_observe scope=full 滚动 sticky 元素发现 (V2.1 已验证 main frame 0 不空)
- V2.1 修复 = D11 动效层观测**完全打开**
